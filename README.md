# 📘 Ebook do Jackson - Portfólio Pessoal

Este projeto é uma página web interativa construída como um portfólio pessoal e um "Ebook" biográfico. O objetivo é apresentar a minha trajetória, desde a infância até a graduação em Análise e Desenvolvimento de Sistemas, utilizando tecnologias fundamentais de desenvolvimento web.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo e das seções de história.
* **CSS3**: Estilização moderna utilizando variáveis (:root), Grid Layout, Flexbox e efeitos de Glassmorphism.
* **JavaScript**: Lógica para alternância de tema (Modo Claro/Escuro) com persistência de dados no navegador.

## 🎨 Funcionalidades Principais

* **Alternador de Temas**: Um botão interativo que permite alternar entre o Modo Escuro (padrão) e o Modo Claro.
* **Persistência de Preferência**: O tema escolhido pelo usuário é salvo no `localStorage`, permanecendo ativo mesmo após recarregar a página.
* **Navegação Suave (Smooth Scroll)**: Menu com links que levam o usuário suavemente até a seção desejada.
* **Design Responsivo**: Layout adaptável para diferentes tamanhos de tela através de Media Queries.
* **Galeria de Fotos**: Uma seção visual organizada com CSS Grid para exibir momentos marcantes.

## 📂 Estrutura do Projeto

* `N2.html`: Arquivo principal com a estrutura semântica.
* `CSS/CSSgeral.css`: Estilização, definições de cores e layouts de grade.
* `Script.js`: Lógica de manipulação do DOM para controle do tema.
* `/IMG`: Pasta contendo as imagens utilizadas no projeto.

## 🛠️ Como Executar o Projeto

1. Faça o download ou clone este repositório.
2. Certifique-se de que a estrutura de pastas (`CSS`, `IMG`) e o arquivo `Script.js` estão no mesmo diretório que o `N2.html`.
3. Abra o arquivo `N2.html` em qualquer navegador moderno.
